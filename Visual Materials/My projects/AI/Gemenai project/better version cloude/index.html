How to use MapRitual

➕

Add a place manually

Tap Menu → Search, type any place name or address. Results appear below — tap "Add as new place" to pin it on your map under the name you typed, even if the suggestions don't match exactly.

If the place can't be found automatically, you can still add it and enter its coordinates by hand.

📥

Import from Google Maps

This is a 4-step process — takes about 5 minutes the first time:

  1. Open Google Takeout — search it in Google or go to takeout.google.com
  2. Click "Deselect all", then scroll down and tick only "Maps (your places)"
  3. Click Next step → Create export. Google emails you a download link (usually within a few minutes).
  4. Download the ZIP file → open it → find the Takeout / Maps folder → you'll see files ending in .json, .csv or .kml / .kmz
  5. Back in MapRitual, tap Menu → Import Places and select those files. Done! 🎉
📊

Importing CSV / spreadsheet files

MapRitual also accepts .csv files — including the "Saved places" lists from Google Takeout, or your own spreadsheet exported as CSV.

For best results, include columns for name/title, and either latitude/longitude or an address (MapRitual will look up coordinates automatically). A URL column is also recognized if present.

📁

Folders & categories

Imported files become folders automatically. Tap a folder to filter the map to just those pins. Each pin can also have a category (Restaurant, Museum…) and a status (Want to Go, Loved It…).

Don't see a category that fits (e.g. "friend's home")? Use the ➕ Add custom category option in settings to create your own, with its own emoji icon.

🎨

Status colours

Each pin's colour shows its status: blue = Want to Go, amber = Been There, green = Loved It, brown = Meh, red = Skip It, grey = Unsorted, and gold = Favourite.

🕐

Recently added

Places you add manually (not imported ones) show up in the Recently added section on the sidebar, so you can quickly find your latest additions.

📌

Edit a place

Tap any pin on the map to open its details. Change the name, category, status or folder, then tap Save.

⚠️

Unpinned places

If MapRitual couldn't find a location for an imported place, it shows up under ⚠️ Unpinned places in the sidebar.

Tap the 📍 Fix All button to look them all up automatically — it changes to ⏳ progress (e.g. "3/12") while it works through the list, about one place per second, so larger lists can take a minute or two.

For anything it can't find, tap the ✏️ next to that place to enter its coordinates by hand.

🗂️

Trip planner & stats

Use Trip planner to turn a folder of places into a day-by-day itinerary by walking or driving. Check My stats for an overview of countries, categories and your travel progress.

🌓

Light & dark theme

Tap the moon/sun icon in the top bar to switch between light and dark mode. Your choice is saved for next time.

⚙️ Backup & Restore

Your data lives only on this device. Save a backup file before switching phones or browsers — and load it again on the other device to bring everything across.

📤 Sharing a folder with a friend: open the folder list, tap 📤 next to any folder to download just that folder as a file. They can open it with "Restore from backup" above (leaving the replace box unchecked) to add it to their own map.
