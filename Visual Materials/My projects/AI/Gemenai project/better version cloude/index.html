How to use MapFolio

➕

Add a place manually

Tap Menu → Search, type any place name or address. Results appear below — tap "Add new place" to pin it on your map.

📥

Import from Google Maps

This is a 4-step process — takes about 5 minutes the first time:

  1. Open Google Takeout — search it in Google or go to takeout.google.com
  2. Click "Deselect all", then scroll down and tick only "Maps (your places)"
  3. Click Next step → Create export. Google emails you a download link (usually within a few minutes).
  4. Download the ZIP file → open it → find the Takeout / Maps folder → you'll see files ending in .json or .kml / .kmz
  5. Back in MapFolio, tap Menu → Import Places and select those files. Done! 🎉
📁

Folders & categories

Imported files become folders automatically. Tap a folder to filter the map to just those pins. Each pin can also have a category (Restaurant, Museum…) and a status (Want to Go, Loved It…).

📌

Edit a place

Tap any pin on the map to open its details. Change the name, category, status or folder, then tap Save.
